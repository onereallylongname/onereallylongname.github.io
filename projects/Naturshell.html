Naturshell

Um projeto de desenvolvimento de produto.

Uma pequena introdução

Naturshell foi um projeto para a disciplina de E.D.P. (Engenharia e desenvolvimento de produto). O objetivo deste projeto era a criação de um produto inovador, relacionado com bicicletas. Este trabalho teve a duração de um semestre, para grupos de quatro pessoas. Para o desenvolvimento do CAD foi utilizado o SolidWorks.

Na moldura em baixo é disponibilizado o relatório do projeto.
